# 贡献指南

感谢您考虑为 DiskScanner 项目做出贡献！

## 提交 Pull Request

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 代码风格

- 遵循 Go 标准代码风格
- 使用 `gofmt` 格式化代码
- 添加适当的注释
- 确保测试通过

## 报告 Bug

请使用 GitHub Issues 报告 bug，并包含：

- 问题描述
- 复现步骤
- 期望行为
- 实际行为
- 系统环境信息
